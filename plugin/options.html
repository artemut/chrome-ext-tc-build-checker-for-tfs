Plugin was tested with TFS 2017 and TeamCity 10.x.

It may not work properly with older versions.

In case of errors, please, open an issue on the plugin's github repository.

TFS Hostname TC Url

Enter the Hostname of the Team Foundation Server where you want to apply the plugin.

Example:

my-tfs.com

Enter the URL of the Team City server that launches builds from the TFS server specified.

Example:

https://my-tc.com
